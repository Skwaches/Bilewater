# Bilehaven

## Summary
This is an attempt at simulating fluids in a 2D space using the SDL library.

## Requirements 
`cmake`

`SDL3` - If vendored, edit the in the [CMakeLists.txt](./CMakeLists.txt) file to point to the location of the directory containing [SDL3](https://github.com/libsdl-org/SDL). Otherwise, set the vendored option to OFF.
